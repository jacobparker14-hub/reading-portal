For IT / deployment: connecting the AI The portal is proxy-ready. To enable live AI question generation, stand up a simple endpoint at /api/claude that accepts POST {prompt, system} and returns {text} from the Anthropic API. Change the single constant PROXY_URL at the top of the script to your endpoint URL. A 15-line Express or Flask server is all that's needed — the API key never touches the browser.
